# my-sound-game
Это история и документация создания моей игры

Я начал с создания репозитория
Создал пустой репозиторий на гит хаб, содержащий README.md
На хосте же я написал в папке с уже сущствующи проектом
git init
git remote add origin https://github.com/egorzolotarev/my-sound-game
Затем я сделал коммит, и запушил, все это с помощью IDE, и тут возникла проблема, ведь я хотел чтобы у меня весь код был в main ветке, а у меня создалась новая ветка master
Чтобы их объединить, я использовал команды
git branch -m master main - переименовал master в main
git fetch origin - скачал содержимое с гита
git merge origin/main --allow-unrelated-histories - склеил две независимые истории (тут создался коммит, который добавлял файлы из внешнего репозитория в локальный)
git push -u origin main - запушил изменения в гит, при этом -u устанавливает связь между локальной и удаленной веткой, чтобы потом пушить без необходимости явно указывать main
